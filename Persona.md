# User Personas

ระบบนี้ถูกออกแบบโดยอ้างอิงจากกลุ่มผู้ใช้งานที่มีแนวโน้มซื้อแผ่นและตลับเกมผ่านช่องทางออนไลน์ เพื่อให้ฟังก์ชันต่าง ๆ ตอบสนองต่อพฤติกรรมและความต้องการของลูกค้าแต่ละประเภท

---

## Persona 1 : นักเล่นเกมทั่วไป (Casual Gamer)

| รายละเอียด | ข้อมูล |
|------------|--------|
| ชื่อ | ณัฐพล วงศ์ชัย |
| อายุ | 22 ปี |
| อาชีพ | นักศึกษา |
| ประสบการณ์ | เล่นเกมเป็นงานอดิเรก |
| ความถี่ในการซื้อ | 1–2 ครั้งต่อภาคการศึกษา |

### เป้าหมาย
- ค้นหาเกมที่ต้องการได้ง่าย
- ซื้อเกมในราคาคุ้มค่า
- ใช้งานเว็บไซต์ได้สะดวก

### Pain Points
- ค้นหาเกมยาก
- ไม่ทราบสถานะสต็อก
- ต้องเปรียบเทียบราคาหลายเว็บไซต์

### ความต้องการ
- ระบบค้นหาและกรองสินค้า
- แสดงจำนวนสินค้าคงเหลือ
- โปรโมชั่นและส่วนลด
- ขั้นตอนสั่งซื้อที่รวดเร็ว

## Persona 2 : นักสะสมเกม (Game Collector)

| รายละเอียด | ข้อมูล |
|------------|--------|
| ชื่อ | ธนวัฒน์ กิตติศักดิ์ |
| อายุ | 30 ปี |
| อาชีพ | โปรแกรมเมอร์ |
| ประสบการณ์ | นักสะสมเกม |
| ความถี่ในการซื้อ | หลายครั้งต่อเดือน |

### เป้าหมาย
- สะสมเกมเวอร์ชันแผ่นและตลับ
- ไม่พลาดเกม Limited Edition
- บันทึกเกมที่สนใจก่อนตัดสินใจซื้อ

### Pain Points
- เกมหายากหมดสต็อกเร็ว
- ลืมเกมที่ต้องการซื้อ
- รายละเอียดสินค้าไม่ครบ

### ความต้องการ
- ระบบ Favorite
- รายละเอียดสินค้าครบถ้วน
- สถานะสต็อกแบบ Real-time
- ระบบติดตามคำสั่งซื้อ

## Persona 3 : ผู้ปกครองที่ซื้อเกมให้บุตรหลาน

| รายละเอียด | ข้อมูล |
|------------|--------|
| ชื่อ | ศิริพร บุญส่ง |
| อายุ | 41 ปี |
| อาชีพ | พนักงานบริษัท |
| ประสบการณ์ | เล่นเกมน้อย |

### เป้าหมาย
- ซื้อเกมที่เหมาะกับลูก
- เลือกเกมได้ง่าย
- ได้รับสินค้าตรงเวลา

### Pain Points
- ไม่เข้าใจแพลตฟอร์มเกม
- เลือกเกมไม่ถูก
- กังวลเรื่องการจัดส่ง

### ความต้องการ
- รายละเอียดสินค้าเข้าใจง่าย
- ระบุแพลตฟอร์มชัดเจน
- ระบบติดตามพัสดุ
- ขั้นตอนสั่งซื้อที่ไม่ซับซ้อน

## Persona 4 : เกมเมอร์สายแข่งขัน (Competitive Gamer)

| รายละเอียด | ข้อมูล |
|------------|--------|
| ชื่อ | ภากร ลิ้มสกุล |
| อายุ | 27 ปี |
| อาชีพ | สตรีมเมอร์ / Content Creator |
| ประสบการณ์ | ผู้เล่นระดับสูง |

### เป้าหมาย
- ซื้อเกมใหม่ทันวันวางจำหน่าย
- ได้รับสินค้ารวดเร็ว
- ไม่พลาดโปรโมชั่น

### Pain Points
- เกมใหม่หมดสต็อกเร็ว
- ขั้นตอนสั่งซื้อหลายขั้นตอน
- โปรโมชั่นไม่ชัดเจน

### ความต้องการ
- ระบบค้นหาที่รวดเร็ว
- Favorite
- โปรโมชั่นที่เห็นได้ชัด
- Checkout ที่รวดเร็ว

# สรุปภาพรวมของ Persona

จากการวิเคราะห์ Persona พบว่ากลุ่มลูกค้าเป้าหมายของระบบมีช่วงอายุประมาณ 18–45 ปี ครอบคลุมตั้งแต่นักศึกษา วัยทำงาน ไปจนถึงผู้ปกครองที่เลือกซื้อเกมให้บุตรหลาน โดยแบ่งออกเป็น 4 กลุ่มหลัก ได้แก่ นักเล่นเกมทั่วไป นักสะสมเกม ผู้ปกครองที่ซื้อเกมให้บุตรหลาน และเกมเมอร์สายแข่งขัน ซึ่งแต่ละกลุ่มมีพฤติกรรมและความต้องการที่แตกต่างกัน ทั้งด้านการค้นหาสินค้า การเปรียบเทียบราคา การเลือกซื้อเกม การติดตามคำสั่งซื้อ และการเข้าถึงโปรโมชั่น อย่างไรก็ตาม ผู้ใช้งานทุกกลุ่มมีความต้องการร่วมกัน คือ ระบบที่ใช้งานง่าย มีข้อมูลสินค้าและสต็อกที่ถูกต้อง ขั้นตอนการสั่งซื้อที่สะดวก และสามารถติดตามสถานะคำสั่งซื้อได้อย่างชัดเจน เพื่อให้ได้รับประสบการณ์การซื้อสินค้าออนไลน์ที่รวดเร็ว น่าเชื่อถือ และตอบโจทย์การเลือกซื้อเกมในแต่ละรูปแบบ
