## ชื่อโครงการ ระบบการขายออนไลน์และสมาชิก (Game Disc and Cartridge E-Commerce System)

### 1. หลักการและเหตุผล (Rationale)

ในปัจจุบันสะดวกความสามารถในการสืบค้นได้ร้องขอการสัง่ซื้อผ่านทางดิจิทัล (Digital Download) เช่นการซื้อเกมผ่านทางแพลตฟอร์มดิจิทัลทำให้ผู้คนจำนวนมากสนใจในการเล่นเกมสด (Retro Games) ที่ใช้อยู่บนคอนโซล (Console) บ้านเก่าแบบเหล่านั้นเพราะว่าเรามีนวัตกรรมใหม่ในการซื้อหาของรสนิยมคลาสสิก ส่วนมากของผู้คนที่สนใจอยู่ในเขตต่างๆ (Zone) และการบริหารจัดการบนแพลตฟอร์ม

บางครั้ง การสั่งซื้อสินค้าออนไลน์บนเว็บไซต์ที่จำหน่ายเกม สามารถที่จะดำเนินการได้ อย่างไรก็ตาม ผลมาจากการว่างของการบริหารจัดการ E-Commerce ที่มีความเชี่ยวชาญเฉพาะสำหรับการขายผ่านอินเทอร์เน็ต ผลมาจากความต้องการในการปรับปรุงระบบเพื่อเพิ่มประสิทธิภาพและการตอบสนองของผู้ใช้

### 2. วัตถุประสงค์ของโครงการ (Objectives)

1. เพื่อให้ระบบเน้อมิลิยิธิสินค้า (E-Commerce) สำหรับการขายและสมาชิก ที่มีประสิทธิภาพและมีอยู่
2. เพื่อให้ระบบการจัดการสินค้า (Product Management) ที่ดีและมีประสิทธิภาพและดูแลสมาชิก
3. เพื่อให้ระบบจัดการคำสั่งซื้อ (Order Management) และการสร้างรหัสของหลักการสัง่ซื้อ (Mock Shopping API) ที่มีประสิทธิภาพ
4. เพื่อให้ระบบควบคุมข้อมูลสินค้า (Revenue Dashboard) สำหรับผู้ดำเนินการ เพื่อให้ติดตามคุณค่าและการบริหารจัดการ
